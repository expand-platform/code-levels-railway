# MVP-scoped concept

Тебе не нужна:
❌ масштабная LMS
❌ идеальный интерфейс
❌ готовность к тысячам

Тебе нужна:
✅ машина дисциплины и стандарта
✅ достаточная для того, чтобы удерживать 5-100 человек одновременно


## Концепт MVP
1. Выдерживает 20-100 одновременных асинхронных студентов без моего участия (только сдача ДЗ в виде milestones / checkpoints - не чаще 1 раза в день, со временем - автоматизация ревью через GPT до необходимого стандарта + после него - уже моё участие, то есть final mentor review)
2. Асинхронный тренажёр по code levels: next part (paginator) заблокирован (isLocked), пока не пройден предыдущий фрагмент (part) 
3. Система контроля качества кода: установление обязательных стандартов, ниже которых нельзя опускаться (отсев слабых, ленивых, немотированых)
4. Мотиватор: повторная сдача, deadlines, очки, трекинг прогресса, уровни, блокировки, экзамены, чекпоинты, goals, milestones
5. Чёткая структура с постановкой целей через теория + задача + проект (1 тема = теория+задача+проект+submission = 1 конкретно выполненная цель (освоить базу - научиться верстать, углубиться в JS - научиться работать с DOM и т.д))
