Маркетинг EXPAND: тихий, спокойный, глубокий, вдумчивый 

- подкаст
- тексты
- имейл-рассылки
- статьи

Теперь главное, определиться:

- о чем писать (новости, обзоры, рецепты, мнение)
- лайфхаки (обучалки, стили, мнения)
- интересные подборки (гитхаб)
- можно брать вдохновение из статей других (devbytes)
- можно свои туториалы / гайды использовать в рекламных целях (то, что хотел делать для Ютуб, делать для статей)
